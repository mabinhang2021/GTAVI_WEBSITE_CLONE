## 任务1: 图片放大按钮（优先级：中）
**文件**: 多个组件（Jason, Lucia, NextJason, NextLucia, NextCal, Leonida, Hotel 等）

**需求**: 为每个带有图片的 div 右下角添加放大按钮，点击可查看大图

**实现方案**:
1. 创建新组件 `ImageViewer` (`src/components/ImageViewer.jsx`)
   - 接收 `src`, `alt`, `className` 等 props
   - 状态管理：isZoomed (boolean)
   - 右下角放置放大按钮图标
   - 点击按钮切换全屏/原尺寸查看模式
   - 使用 Portal 将放大层渲染到 body
   - ESC 键关闭

2. 需要修改的组件:
   - `Jason.jsx`: jason-1, jason-2, jason-3
   - `Lucia.jsx`: lucia-1, lucia-2, lucia-3
   - `NextJason.jsx`: next-jason-1, next-jason-2, next-jason-3, next-jason-4
   - `NextLucia.jsx`: next-lucia-1, next-lucia-2, next-lucia-3
   - `NextCal.jsx`: next-cal-1, next-cal-2, next-cal-3, next-cal-4, next-cal-5
   - 其他可能有图片的组件

3. 新增 CSS 样式到 `src/index.css`

## 任务3: 抽象可复用组件（优先级：低）
**文件**: 新建 `src/components/CharacterCard.jsx`

**需求**: 抽象 Cal, NextCal, PostCard 的共同模式为可复用组件

**分析三个组件的共同模式**:
- `Cal`: 文字内容 + 背景动画
- `NextCal`: 网格布局图片 + 视频 + 文字叠加
- `PostCard`: 单卡片样式 + 视频 + 渐变背景

**初步设计的通用接口**:
```jsx
<CharacterCard
  variant="grid" | "single" | "card"  // 不同展示变体
  images={[{ src, alt, className }]}  // 图片数组
  video={{ src, className }}           // 可选视频
  title=""                             // 标题
  subtitle=""                          // 副标题
  description=""                       // 描述文字
  quote=""                             // 可选引用
  textPosition="left" | "right"        // 文字位置
/>
```

**注意**: 这个抽象需要仔细设计，建议放在最后做，以确保接口设计合理
